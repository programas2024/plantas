Acerca de PlantID

PlantID es una aplicación que utiliza inteligencia artificial para identificar plantas medicinales y mostrar información sobre sus propiedades, usos tradicionales y precauciones.

Cómo funciona

  1. Selecciona una imagen de una planta desde tu galería o toma una foto
  2. Nuestro sistema analizará la imagen usando un modelo de machine learning
  3. Te mostraremos información detallada sobre la planta identificada

Tecnología

Esta aplicación utiliza TensorFlow.js y un modelo entrenado con Teachable Machine para la identificación de plantas.

Safire

Safire es una funcionalidad premium que te permite:

  • Identificar un catálogo más amplio de plantas
  • Obtener información más detallada sobre usos medicinales
  • Recibir recomendaciones personalizadas según tus necesidades
  • Acceder a recetas tradicionales con las plantas identificadas
